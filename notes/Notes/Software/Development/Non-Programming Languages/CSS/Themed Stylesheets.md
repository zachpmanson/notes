A list of stylesheets that emulate different kinds of interfaces.

- [TheSims.css](https://thesimscss.inbn.dev/), for emulating The Sims
- [98.css](https://jdan.github.io/98.css/), for emulating Window 98
- [95.css](https://alexbsoft.github.io/win95.css/), for emulating Window 95
- [XP.css](https://botoxparty.github.io/XP.css/), for emulating Windows XP
- [7.css](https://khang-nd.github.io/7.css/), for emulating Windows 7
- [cs16](https://cs16.samke.me/) counter strike 1.6
- [tufte-css](https://edwardtufte.github.io/tufte-css/) edward tufte
- [PSone.css](https://micah5.github.io/PSone.css/) playstation one
- [NES.css](https://nostalgic-css.github.io/NES.css/) nes
- [System.css](https://sakofchit.github.io/system.css/) apple system
