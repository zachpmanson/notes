Fediverse Family Tree

For keeping track of software forks in the Fediverse, and why they exist.

  • Misskey
    • Calckey (later renamed to the equally bad name, firefish), more feature rich?
      • IceShrimp, Yet Another Misskey Fork with the "goal of increasing stability and performance"
    • Sharkey, generally follows upstream, implements the Mastodon API
  • Pleroma
    • Akkoma, "is a specialised fork of Pleroma. It is faster-paced in development and aligns itself more closely with Misskey than Pleroma mainline"
  • Mastodon
    • glitch-soc , a friendly fork of Mastodon with a large number of experimental features, appears to follow upstream
    • hometown, a gentle fork of Mastodon wanting to add a small number of features and generally stay close to upstream (though its a few thousand commits behind)
  • Lemmy, a reddit-like fediverse server
  • Kbin, a reddit-like fediverse server with some masto-style microblogging aspects? It is interoperable with Lemmy allegedly
    • Mbin, a "community focused" fork of mbin that will. Is apparently much more open to community input and PRs.
  • Goblin, a tumblr style ActivityPub server
  • Wafrm, another tumblr style ActivityPub server
