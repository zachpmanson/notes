Elk

Elk is a web client for Mastodon that I prefer to the default frontend. It is open source and is built with Nuxt. It also supports glitch-soc (a Mastodon fork) and GoToSocial (a lightweight ActivityPub server).

I use a patched version of Elk with a few of my own tweaks at elk.zachmanson.com.
